The following CSS classes allow you to control typography. Most of them are available in responsive variants.

### Alignment
Use to alignment inline elements
- **Responsive:** No

{% highlight scss %}
.ta-left,
.tl {
  text-align: left;
}
.ta-center,
.tc {
  text-align: center;
}
.ta-right,
.tr {
  text-align: right;
}
{% endhighlight %}
