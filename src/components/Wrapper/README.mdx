Wrapper can wrap it's content in a specific width depending on if the application has a full or fixed width.

```js
import { Wrapper } from '@wfp/ui';
```

```js
<Wrapper pageWidth="xs" mobilePageWidth="full">
  Content
</Wrapper>
```
