<br /><br /><h2>Sketch</h2>


<p>
  To get started with the UI Kit library, you must first have <a href="https://www.sketch.com" target="_blank">Sketch</a> installed. The UI Kit components and icons live in the design kit as Sketch symbols: <strong>the symbols are organized by component, and every component comes with its variations, states, and groups</strong>.
</p>
